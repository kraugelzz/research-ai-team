📖 คู่มือใช้งานเบื้องต้น

วิธีใช้งาน + ความหมายของไอคอนทุกตัวในเว็บ

🚀 เริ่มใช้งานใน 4 ขั้น

พิมพ์คำค้นเป็นภาษาอังกฤษ
เช่น ANK3 renal cell carcinoma — หรือคลิกตัวอย่างใต้ช่อง หรือลากไฟล์ .txt มาวาง
ตั้งค่า (ถ้าต้องการ)
เลือกวิธีเรียง / ตั้งแต่ปี / จำนวนเปเปอร์ / ตัวกรองคุณภาพ
กดปุ่ม ▶ Run ทีม
ทีมงาน 6 คนทำงานต่อกัน: ค้นหา → ตรวจสอบ → สกัด → เช็ก → สรุป → แปลไทย
อ่านผล + คัดกรอง + ดาวน์โหลด
กดปุ่มคัดกรองแต่ละเปเปอร์ (บันทึกอัตโนมัติ) แล้ว export เป็น CSV / RIS / PRISMA

🔀 โหมดการเรียงผลลัพธ์

✨ อัจฉริยะ
ตรงคีย์เวิร์ดเป็นหลัก + ดันเปเปอร์ปีล่าสุดขึ้นมา (แนะนำ — ค่าเริ่มต้น)
🎯 ความเกี่ยวข้อง
ตรงคีย์เวิร์ดที่สุด ไม่สนใจปี
🆕 ใหม่สุดก่อน
เรียงตามวันที่ตีพิมพ์ล้วน (อาจมีนอกเรื่องปนบ้าง)
⭐ อ้างอิงมากสุด
งานคลาสสิกที่ถูกอ้างอิงเยอะ (มักเป็นงานเก่า)

🚩 ธงเตือนคุณภาพงาน

⛔ RETRACTED
งานถูกถอน — ห้ามอ้างอิง! (ระบบซ่อนให้อัตโนมัติ เปิดดูได้ในตัวกรอง)
📄 preprint
ยังไม่ผ่าน peer review — ใช้อย่างระวัง ยังไม่ถือว่าตรวจสอบแล้ว
📚 review
บทความรีวิว ไม่ใช่งานทดลองต้นฉบับ (ตัดออกได้ในตัวกรอง)

🧪 ชิปข้อมูลสกัด (คอลัมน์ "สกัด")

ระบบดึงข้อมูลสาย wet-lab จากบทคัดย่ออัตโนมัติ แยกด้วยสี

shRNA
สีน้ำเงิน = วิธีทดลอง — siRNA / shRNA / CRISPR / knockout / overexpression
786-O
สีเขียว = cell line — เซลล์ที่ใช้ในงาน (คลังกว่า 90 สาย)
human
สีส้ม = สิ่งมีชีวิต — human / mouse / rat / zebrafish

✅ ปุ่มคัดกรอง (คอลัมน์ "คัดกรอง")

กดเพื่อจัดสถานะแต่ละเปเปอร์ — บันทึกในเครื่องอัตโนมัติ เปิดใหม่ก็ยังอยู่

✓
รับเข้า (include) — เปเปอร์ที่เข้าเกณฑ์ ใช้ในรีวิว (แถวขึ้นสีเขียว)
?
รอพิจารณา (maybe) — ยังไม่แน่ใจ ไว้กลับมาดู
✗
ตัดออก (exclude) — ไม่เข้าเกณฑ์ (แถวจางลง) · ใส่เหตุผลในช่องโน้ตได้

🔗 ลิงก์แหล่งข้อมูล (คอลัมน์ "ลิงก์")

DOI↗
ไปหน้าทางการของเปเปอร์ (สำนักพิมพ์)
PubMed↗
หน้า PubMed พร้อมบทคัดย่อและข้อมูลอ้างอิง
FullText↗
ฉบับเต็มบน Europe PMC (ถ้ามี)
PDF↗
โหลด PDF ฉบับเต็มฟรี — เฉพาะงาน open access
OA
Open Access — อ่านฉบับเต็มได้ฟรี ไม่ติดค่าสมาชิก

👥 ทีมงาน 6 คน (แต่ละขั้นของสายพาน)

🔍 สกาวต์
ค้นหาเปเปอร์จาก Europe PMC
✅ วาลิดา
ตรวจสอบ กันซ้ำ กรองคุณภาพ (retracted/review)
📊 แอนนา
สกัดเป็นตาราง + ดึงข้อมูล wet-lab
⚖️ เชค
จัดลำดับหลักฐาน + หาผลที่ขัดแย้งกัน
✍️ ไรเตอร์
สังเคราะห์สรุปจากบทคัดย่อจริง (ไม่กุ)
🌐 ทรานส์
แปลอังกฤษ→ไทย + สรุปว่าเปเปอร์ทำอะไร

ดูรายละเอียดตัวละครแต่ละคน + เครื่องมือจริงได้ที่ หน้าแนะนำทีม →

⬇️ ปุ่มดาวน์โหลด (Export)

CSV
ตารางเปเปอร์ — เปิดใน Excel
BibTeX / RIS
นำเข้า Zotero / EndNote / Mendeley ได้เลย
Markdown
สรุปสังเคราะห์ พร้อมลิงก์อ้างอิง
PRISMA log
บันทึกการค้นหาแบบทำซ้ำได้ + ประโยค Methods สำเร็จรูป ใส่ paper ได้
ผลคัดกรอง
CSV รวมสถานะ include/exclude + เหตุผล + ข้อมูลสกัด

💡 เคล็ดลับ

พิมพ์ภาษาอังกฤษ — ฐานข้อมูลงานวิจัยเป็นอังกฤษเกือบทั้งหมด พิมพ์ไทยจะหาไม่ค่อยเจอ
เจอน้อยไป? ตัดคำให้กว้างขึ้น เช่น ANK3 renal cell carcinoma → ANK3 kidney cancer
อยากได้ทั้งใหม่ทั้งตรง ใช้โหมด "✨ อัจฉริยะ" + ใส่ช่อง "ตั้งแต่ปี 2023"
โควตาแปลหมด? รอวันถัดไป หรือปิด "🌐 แปลไทย" แล้วยังค้นได้ปกติ (การแปลใช้บริการฟรีที่มีลิมิตรายวัน ไม่กิน token ใดๆ)
สรุปเป็นแบบสกัด — ทุกประโยคดึงจากบทคัดย่อจริง ไม่มีการกุ แต่ควรเปิดอ่านต้นฉบับก่อนอ้างอิงเสมอ
